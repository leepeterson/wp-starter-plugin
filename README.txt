== Description ==

This is a starter plugin used for creating other plugins. This should be the starting point for creating other
plugins and a repository of techniques gathered from various plugins and adapted for use within the
WordPress Plugin Boilerplate



== Installation ==

1. Clone this plugin.
2. Rename everything for the new plugin.
3. Create said new plugin.
4. Install said new plugin in the wp-content/plugins folder.
5. Activate the new plugin.



== Frequently Asked Questions ==

= What does this plugin do? =

Nothing helpful other than show methods for how to create plugin parts.
This plugin should not used in production, this only serves as an example.
